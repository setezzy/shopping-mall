# shopping-mall

E-shop: SSM (Spring + SpringMVC + MyBatis) + Maven

- 开发环境

Intellij IDEA, JDK 1.8, Tomcat 8.5.42

- 后端技术

核心框架：Spring Framework 5.1.5

视图框架：Spring MVC 5.1.5

持久层框架：MyBatis 3.4.6

数据库：MySQL 8.0.12

数据库连接池：DBCP 1.4

- 前端技术

JS框架：Jquery 3.2.0

- 其他

项目管理：Maven 4.0.0

日志管理：slf4j 1.7.25 + log4j 2.11.1

单元测试：Junit 4.12
